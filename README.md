# FastAPI - QrApp
Generating QrCode from a api call, using FastApi python

## Start API
### 1. Create & Activate vritual env
  ```bash
  > python -m venv .
  > source ./bin/activate
  ```
### 2. Install Require packages
  ```bash
  > python -m pip install requirments.txt
  ```
### 3. Run Module
  ```bash
  > cd API
  > python main.py
  ```
